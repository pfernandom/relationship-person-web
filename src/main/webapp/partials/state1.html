Accenture's Link Tracker Micro-Service


Welcome to ALT-MS, Accenture's Link Tracker Micro-Service, a micro-service to define and query.
Our graph database (Neo4J) creates and stores relationships between nodes (Persons). Users of the application (case workers, data analysts) can use this information to find relationships that might be hidden in the data.
